Component API / Options
Option Value Description
art-joystick Object The main directive declaration.
send-movement Callback will callback one of: {response: {left: speed, right: speed}}, {response: {angle: 27, speed: speed}}. {response: {direction: 'up-left', speed: speed}}
cordova-enabled Boolean If enabled, find the cordova extension and run the events through that
vibration-enabled Boolean If enabled, try to find some vibration controls and run then
speed-increase String 'linear', 'exponential'
max-positive-speed Integer Maximum positive (forward) speed
min-positive-speed Integer Maximum negative (backward) speed
direction-management String 'differential' - means we send acceleration for each wheel through the callback (real devices); 'angle' - we send the angle of movement and acceleration through the callback (real / on-screen interactive element); 'directions': - we send 'up', 'up-left', 'up-right', 'left', 'right', 'down-left', (on-screen interactive element); 'down-right' and acceleration through the callback
control-size Integer The size in pixels of the control on page, max 300
